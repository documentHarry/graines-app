Graines de folie

Application de gestion d'un catalogue de graines de plantes.

Cette première version permet de lire les données depuis une base SQLite grâce à Electron, Prisma et Angular.

Fonctionnalités disponibles

  • Afficher les catégories
  • Afficher les produits du catalogue
  • Consulter la fiche détaillée d'un produit
  • Lire les données depuis SQLite
  • Utiliser une navigation Angular simple

Architecture utilisée

L'interface est réalisée avec Angular. L'accès aux données passe par Electron, puis par Prisma, afin de lire la base SQLite locale.
